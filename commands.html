Teleport, homes & warps

Command What it does
/spawn Teleport to the world spawn.
/home [name] Go to a saved home; omit name for the default home.
/sethome [name] Save your current location as a home.
/delhome [name] Remove a home. /delhome all removes every home.
/homes List your homes (same as /homelist).
/tpa <player> Ask to teleport to another player.
/tpahere <player> Ask another player to teleport to you.
/tpaccept [user] Accept a pending teleport request.
/tpdeny [user] Decline a request (also /tpdecline, /tpno).
/rtp Random teleport (arguments depend on server config).
/warp [name] Teleport to a warp. /warps / /warplist lists them.
/back Return to your previous location (e.g. after death or teleport).

Economy & shop

Command What it does
/pay <player> <amount> Send currency to another player.
/bal [player] Show balance (yours or another player’s if allowed).
/baltop [page] Leaderboard of top balances.
/paytoggle Toggle whether you can receive /pay.
/shop [section] Open the server shop; optional section name.
/sell Sell items by value. For bulk sales use /sellall or /sellgui.
/sellall · /sellall <material> · /sellall hand Sell items from your inventory.
/sellgui Open the graphical sell menu.

Chat & messaging

Command What it does
/msg <player> <message> Private message (aliases often include /tell, /m, /whisper).
/r <message> Reply to the last player who messaged you.
/ignore <player> Hide chat from a player.
/ignorelist List players you’re ignoring.

Auction house

Command What it does
/ah Open the auction house.
/ah search <filter> View items matching the search filter.
/ah help Show help for all auction commands.
/ah sell <price> Sell the item you are holding in your hand.
/ah selling View the items you are selling.
/ah sold View items you have sold recently.
/ah expired View your expired items to return.
/ah cancel Cancel all your auctions.
/ah return Return all your cancelled or expired items.

Scoreboard & info

Command What it does
/sb Toggle the sidebar scoreboard. Requires permission.
/help Server help and command list.
/rules Display server rules.
/discord Shows the Discord invite in chat.
/vote Server voting links and rewards.
/suicide Eliminate your character.
