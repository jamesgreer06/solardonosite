Fair play

Prohibited mods & clients

Java and Bedrock already play on different clients—we don’t need Java-only tools that make navigation and awareness easier than what Bedrock players can match. These mods are banned because they’re common on the server and widen that gap.

Explicitly not allowed

Do not use the following (including optional features like cave maps or entity radar if you enable them):

Mod Why
Xaero’s World Map Full map and waypoint tooling not available to Bedrock; unfair world awareness at scale.
Xaero’s Minimap Minimap and related features (waypoints, radar-style info) stack advantage on Java over Bedrock.

Everything else

Hacks, x-ray, automation that bypasses fair play, and other unfair edges are still covered under the server rules. If you’re unsure about a mod or shader pack, ask staff on Discord before you rely on it.
